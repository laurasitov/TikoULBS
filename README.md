# Tiko – ULBS Assistant

Tiko is the official AI assistant for Universitatea “Lucian Blaga” din Sibiu (ULBS). This Android application provides students with a convenient way to get answers to their questions about university life.

## Features

- **AI-Powered Chat**: Ask Tiko questions in natural language and get helpful answers.
- **ULBS Branding**: The app is styled with the official ULBS color palette.
- **Local FAQ Lookup**: Common questions are answered instantly from a local database.
- **Conversation History**: Your chat history is saved on your device for future reference.

## Getting Started

1.  **Clone the repository**.
2.  **Add your OpenAI API key** to your `local.properties` file:

    ```
    OPENAI_API_KEY="YOUR_API_KEY"
    ```

3.  **Build and run** the application in Android Studio.

## Disclaimer

Tiko provides information based on publicly available data. Always verify critical information with official ULBS sources.
